# **Custom Sensor Design for Real-Time Water Leak Detection**

This repository contains the 3D model of a custom-designed sensor developed for real-time water leak detection in urban distribution networks. The sensor is an integral component of our IoT-based framework, combining innovative hardware and data-driven analytics to enhance the efficiency, scalability, and accuracy of water management systems.

---

## **Overview**

Water distribution networks face increasing challenges from undetected leaks, leading to resource wastage and operational inefficiencies. This sensor design addresses these challenges by facilitating precise monitoring and anomaly detection. The sensor integrates seamlessly with edge computing devices and supports real-time data processing for proactive leak identification and resolution.

---

## **Key Features of the Design**

1. **Compact and Modular Structure**  
   - The design ensures ease of deployment in diverse network configurations.  
   - Modular components allow for scalability and integration with various IoT platforms.

2. **Enhanced Durability**  
   - Constructed with high-quality materials, the sensor is suitable for harsh environmental conditions typical of water distribution systems.

3. **Optimized for Real-Time Processing**  
   - Supports local data preprocessing using physics-informed machine learning models to filter noise and provide actionable insights.

4. **Adaptive Connectivity**  
   - Features built-in compatibility with low-power communication protocols such as LoRa and NB-IoT for seamless integration into existing infrastructure.

## **Integration Workflow**

1. **Sensor Deployment**  
   Deploy the sensor at optimal locations in the network, as identified using the Partial Set Network Length Covering Location Problem (PSNLCLP) algorithm.

2. **Data Collection**  
   The sensor collects real-time pressure and flow data from the network, transmitting preprocessed information to the edge computing device.

3. **Data Analysis**  
   - Physics-informed machine learning models filter noise and detect anomalies.  
   - The centralized system aggregates the processed data, identifying potential leaks and generating actionable insights.

4. **Predictive Maintenance**  
   The data enables proactive maintenance strategies, significantly reducing water loss and repair costs.
